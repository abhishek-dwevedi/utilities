# Markdown Formatting Example

## Header

```
Put Hash or Pound Sign # before text to make them header, one hash for Heading 1 .

# H1
## H2
### H3
#### H4
#### H5
```

## Emphasis

```
** bold **

```

## Blockquotes
```
Put > before text to make it block
Syntax:
> This is example block
>
> Block can be used to define sections
> Highlight text,
```
